# Sokoban Puzzle

## 프로젝트 소개

본 프로젝트는 고전 퍼즐 게임인 Sokoban을 기반으로 제작한 타일 기반 퍼즐 게임입니다.  
플레이어는 맵 위에서 상자를 밀어 장애물들과 상호작용하며 스테이지를 클리어하게 됩니다.

본 프포젝트는 단순한 게임 구현을 넘어,  
객체지향 설계(OOP)와 확장 가능한 구조 설계에 중점을 두고 개발되었습니다.

특히 다음과 같은 부분을 목표로 설계했습니다.
- 역할과 책임을 분리한 객체지향 구조
- 유지보수와 기능 확장을 고려한 클래스 설계
- 유연한 스테이지 구성
- 재사용 가능한 게임 로직 설계

## 개발자 정보

- 이름: 양정훈
- 언어 : Java(Swing, 컬렉션 프레임워크)
